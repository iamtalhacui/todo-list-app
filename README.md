# ✅ To-Do List Web Application

This is a simple and interactive To-Do List Web Application developed as part of the Web Development Virtual Internship (Month 1).

The application allows users to manage daily tasks efficiently.

---

## 🚀 Live Demo

🔗 [View Live App](https://todolist-app-mtalha.vercel.app)

---

## 📌 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Stores tasks using Local Storage
- Responsive design

---

## 🛠️ Technologies Used

- React
- Tailwindcss
- Local Storage
- Vercel (Deployment)

---

## 🎯 Purpose of the Project

The goal of this project was to:

- Practice DOM manipulation
- Understand event handling in JavaScript
- Work with browser Local Storage
- Build an interactive front-end application
